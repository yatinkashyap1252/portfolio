Chattie

🛠️ React • MongoDB • Express.js • Node.js • WebSockets

📖 Description

  • 💬 Real-time messaging app built as a full-fledged alternative to WhatsApp using the MERN stack.
  • ⚡ Instant chat experience with WebSocket technology ensuring seamless, lag-free communication.
  • 🔒 End-to-end encryption for securing conversations and protecting user privacy.
  • 📷 Media sharing with support for images, videos, and document uploads.
  • 👥 Group chat functionality allowing users to create and manage group conversations effortlessly.
  • 📡 Live notifications for new messages, friend requests, and other interactions.
  • 🎨 Custom animated UI with smooth transitions and an intuitive design.
  • 🛠️ Scalable backend powered by Node.js and MongoDB, handling high-traffic loads efficiently.
  • 📱 Mobile-responsive design ensuring compatibility across all devices.
