GlobePath

🛠️ React Native • Firebase • Gemini

📖 Description

  • 🧳 AI-powered travel planner that curates personalized itineraries based on user preferences.
  • 🌍 Customized trip planning considering destination, budget, duration, and travel type (solo, family, couples, friends).
  • 📊 Data-driven recommendations fetched and stored using Firebase for seamless user experience.
  • 🤖 Gemini AI integration to enhance itinerary suggestions with smart insights and optimize travel schedules.
  • 🗺️ Attraction and accommodation recommendations based on user interests and budget constraints.
  • 🎟️ Real-time booking assistance for hotels, flights, and local experiences through API integrations.
  • 📡 Live updates using WebSockets to provide alerts on weather, flight changes, and travel advisories.
  • 📱 Mobile-optimized interface ensuring a smooth, responsive experience on all devices.
  • 🔄 Trip history and reusability allowing users to save and modify past itineraries for future travel.
